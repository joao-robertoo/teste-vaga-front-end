# E-commerce Vitrine de Produtos - Econverse

Projeto front-end para a vaga de Desenvolvedor Front-End Jr na Econverse.

## 🎯 Especificações

- **Framework**: React 18 com TypeScript
- **Build Tool**: Vite
- **Styling**: SCSS (Sass)
- **API**: Consumo de JSON com dados de produtos
- **Design**: Responsivo (Mobile, Tablet, Desktop)

## 🚀 Como Começar

### Pré-requisitos
- Node.js 18+ instalado
- npm ou yarn

### Instalação

```bash
# 1. Instalar dependências
npm install

# 2. Iniciar servidor de desenvolvimento
npm run dev

# 3. Build para produção
npm run build

# 4. Preview do build
npm run preview
```

## 🎨 Features

- ✅ Listagem de produtos em grid responsivo
- ✅ Cards de produtos com informações de preço e desconto
- ✅ Modal interativo ao clicar no produto
- ✅ Controle de quantidade no modal
- ✅ Consumo de API JSON para dados de produtos
- ✅ Design responsivo (mobile-first)
- ✅ HTML semântico
- ✅ SEO friendly
